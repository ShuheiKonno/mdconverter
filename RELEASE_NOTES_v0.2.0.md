## ✨ 新機能

- **PDF をページ毎に構造化** — 各ページ先頭に `## ページ N` 見出しを自動挿入。AI サービスがページ単位で引用・参照しやすくなります
- **出力先フォルダを記憶** — 前回指定した出力先を `~/.mdconverter.json` に保存、次回起動時に自動復元
- **「開く」ボタン** — 出力先フォルダを OS ファイルマネージャ (エクスプローラ / Finder / xdg-open) でワンクリックで開く
- **ページ毎の進捗ログ** — 重い PDF でも処理が生きていることをログ欄で確認できます
- **バージョン表示** — タイトルバーとウィンドウ右下にバージョン番号を表示

## 🐛 バグ修正

- **大容量 PDF (~300 ページ超) で GUI が固まる問題を修正** — ワーカースレッド終了時にイベントキューを完全にドレインしないバグを修正
- **pdfplumber のハング問題を回避** — PDF バックエンドを **pypdfium2** (Google PDFium / Chrome 採用エンジン) に切り替え、複雑な PDF での無限ループを解消。処理速度も 1〜2 桁向上

## 💥 挙動変更

- **画像データを Markdown から除外** — 従来は base64 data URI として埋め込んでいましたが、AI サービス (Claude / ChatGPT / NotebookLM) や RAG パイプラインでは data URI 画像が無視されトークンを無駄にするだけだったため廃止しました。画像の内容を AI に理解させたい場合は、元の PDF / PPTX を直接アップロードしてください
- 「開く」ボタンで出力先欄にファイルパスが入力されていた場合は明示エラーを表示 (任意ファイル起動の防止)

## 📦 ダウンロードと起動

- **`mdconverter.exe`** (約 120 MB) は Windows 10 / 11 用の単体実行ファイルです。Python 環境は不要
- ダウンロード後ダブルクリックで起動
- **初回起動時の注意**: Windows SmartScreen が「不明な発行元」として警告を出す場合があります。「詳細情報」→「実行」で起動してください (未署名バイナリのため)

## ⚠️ 既知の制限

- **スキャン PDF** (画像としての PDF) はテキスト抽出不可 (OCR 未対応)
- **音声ファイル変換** には別途 `ffmpeg.exe` を PATH または実行ファイルと同じフォルダに配置する必要があります
- **画像内容の認識は行いません** — 画像付き PDF / PPTX を LLM に読ませたい場合は、本ツールを経由せず元ファイルを AI サービスに直接投入してください
